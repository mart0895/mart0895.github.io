  15. Qus: Create CSS animation block to rotate an element that has the following properties: Class "rotate-me" and it rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin".

    Ans:
    <html>
    <head>
    <style>
    .rotate-me {
    width: 50px;
    height: 50px;
    background: purple;
    animation: spin 5s infinite;
    }

    @keyframes spin{
    0% {tranform: rotate(72deg); transition-timing-function: linear;}
    25% {transform: rotate(144deg); transition-timing-function: linear;}
    50% {transform: rotate(216deg); transition-timing-function: linear;}
    75% {transform: rotate(288deg); transition-timing-function: linear;}
    100% {transform: rotate(360deg); transition-timing-function: linear;}
    }
    </style>
    </head>
    <body>

    <div id="rotate-me" class="rotate-me"></div>

    </body>
    </html>
