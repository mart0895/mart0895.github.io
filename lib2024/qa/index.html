1. Qus: "What are the three components of Web?"

  2. Qus: "Define version control and give one example."

    Ans: "Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. One example would be Github."

  3. Qus: "What tag is used to create the title of a web page? Give an example."

    Ans: "The <title> tag is what is used to create the title of a webpage. Example: <title>Example 1</title>."

  4. Qus: "What tag is used to create a paragraph in html? Give an example."

    Ans: "The tag used to create a paragraph in html is <p>. Example: <p>Example 2</p>"

  5. Qus: "Identify the odd one from these tags: <span&g;, <p>, <a>, <i> and state the reason?"

    Ans: "The odd one from those tags is <a>, because it's the only inline element out of all the tags."

  6. Qus: "What are the tags required to create a navigation list in HTML?"

    Ans: "The <nav> tag is what is used to create a navigation list in HTML."

  7. Qus: "What element is used to create images? Names its attributes."

    Ans: "The <img> element is what is used to create images, the attributes that are associated with are src= , alt= , width= , height= ."

  8. Qus: "Identify the odd one from these tags: <tags>, <tr>, <b>, <td> and state the reason."

    Ans: "The <b> tag is the odd one out, the reason being is that it's the only one not used for tables in HTML."
